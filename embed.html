Legal System Visualization - Embed Examples

Example 1: Basic Embed (Diagram View)

This example shows a basic embed with default settings (diagram view, filters enabled, read-only mode).

Code:

<div id="my-viz" style="width: 100%; height: 700px;"></div>
<script src="data.js"></script>
<script src="helpers.js"></script>
<script src="network-diagram.js"></script>
<script src="embed.js"></script>
<script>
  LegalSystemEmbed.init({
    container: '#my-viz',
    view: 'diagram',
    showFilters: true
  });
</script>

Example 2: Table View with Filters

This example shows the table view with filters enabled.

Code:

LegalSystemEmbed.init({
  container: '#my-viz',
  view: 'table',
  showFilters: true
});

Example 3: Diagram View without Filters

This example shows the diagram view without filter controls.

Code:

LegalSystemEmbed.init({
  container: '#my-viz',
  view: 'diagram',
  showFilters: false
});
